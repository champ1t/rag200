----------- -- คำสั่งเพิ่ม range-mode -- -------------------

shutdown port ก่อนทำการปรับระยะทาง

(config-if)##range-mode 50 250

50 = 50x100 = 5km

250 = 250*100 = 25Km

(config-if)##range-mode 0 200

0 = 0x100 = 0km

200 = 200*100 = 20Km

(config-if)#show gpon olt range-mode gpon-olt_1/2/3

หลังจากปรับค่าเสร็จให้ทำการ Reset-card

----------- -- คำสั่ง port mirror zte -- -------------------

Mirror for V3：

ethdriver_test -setmiror [direction:0->rx,1->tx,2->rx/tx] [desport] [action：0->disable,1->enable]

example：

ethdriver_test -setmiror 2 0 1       //Mirror wan packets to eth0

ethdriver_test -setmiror 2 1 1       //Mirror wan packets to  eth1

ethdriver_test -setmiror 2 0 0       //Close mirror

（3）、Mirror for V5：

ifconfig mirror add pon0 eth0    //mirror wan to eth0

ifconfig mirror del pon0         //delete mirror
