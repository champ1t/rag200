1.การ command DSL-VPN สำหรับ Forth-Mini

 

ตัวอย่าง กรณีมีการ stack 2 ตัว และจะ config ลูกค้า วงจร 7454ID004 ที่ตำแหน่ง slot 2 port 40 vlan 682

 

>>set to slot1

create vlan static vlanname  7454ID004_vpn vlanid 682 egressports 385 386 untaggedports none  bridgingmode restricted

>>set slot2

create vlan static vlanname 7454ID004_vpn vlanid 682 egressports 40 385 untaggedports 40 bridgingmode restricted

>>map vlan to prot

modify gvrp port info  portvlanid 682 ingressfiltering true portid 40

**คำสั่งดูสถานะ port

$ get gvrp port info portid 1

$ get adsl line intf ifname dsl-36

$ get bridge port intf portid 36

2.การ command Learning disable port 385
