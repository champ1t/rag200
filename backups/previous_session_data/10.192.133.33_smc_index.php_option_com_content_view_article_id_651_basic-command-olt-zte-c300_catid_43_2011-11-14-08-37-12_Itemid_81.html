OL01#config t
Enter configuration commands, one per line.  End with CTRL/Z.
OL01(config)#pon-onu
OL01(config)#pon-onu-mng gpon-onu_1/4/1:2
OL01(gpon-onu-mng)#interface eth eth_0/3 state unlock
