เพิ่ม ip pool ให้ BNG

Bras HYI Redback  - เข้า context fttxcyber

[fttxcyber]hyi_se1200_01#sh ip pool

[fttxcyber]hyi_se1200_01#sh config

interface ip-pool-04-pkt multibind

ip address 1.2.186.129/25

ip pool 1.2.186.128/25

!

หยิบ pool ที่ต้องการออกมา แล้วเก็บข้อมูลไว้ ลบ interface นั้นทิ้งไป

[fttxcyber]hyi_se1200_01(config)#context fttxcyber

[fttxcyber]hyi_se1200_01(config-ctx)#no interface ip-pool-04-pkt multibind

เข้า BNG เพื่อ config poolเข้าไป โดยทำ 4 step

//--------------------------- step 1 ---------------------//

1. show run เพื่อ ดู interface loopback ลำดับถัดไป และสร้าง interface pool ที่เป็น gateway ขึ้นมา

interface Loopback1012

ipv4 address 1.10.150.1 255.255.255.0

!

RP/0/RSP0/CPU0:hyi-bng-01(config)#int loopback 1013

RP/0/RSP0/CPU0:hyi-bng-01(config-if)#ipv4 address 1.4.159.1 255.255.255.0

RP/0/RSP0/CPU0:hyi-bng-01(config-if)#commit

!

//--------------------------- step 2 ---------------------//

2 edit prefix ชื่อ BNG-POOL โดยเรียง ip addres ลำดับตัวเลข

RP/0/RSP0/CPU0:hyi-bng-01#edit prefix-set BNG-POOL

เพิ่ม prefix เข้าไป กด ctrl + X กด yes

!

//--------------------------- step 3 ---------------------//

3. เพื่อ routing OSPF

 

RP/0/RSP0/CPU0:hyi-bng-01(config)#router ospf 23969

RP/0/RSP0/CPU0:hyi-bng-01(config-ospf)#summary-prefix 1.4.159.0/24 tag 64094237

RP/0/RSP0/CPU0:hyi-bng-01(config-ospf)#commit

!

//--------------------------- step 4 ---------------------//

4. ใส่ ip pool เข้าไปให้ลูกค้ารับ

 

RP/0/RSP0/CPU0:hyi-bng-01(config)#pool vrf default ipv4 FTTXCYBER

RP/0/RSP0/CPU0:hyi-bng-01(config-pool-ipv4)#address-range 1.4.159.2 1.4.159.254

RP/0/RSP0/CPU0:hyi-bng-01(config-pool-ipv4)#commit

RP/0/RSP0/CPU0:hyi-bng-01(config-pool-ipv4)#exit

RP/0/RSP0/CPU0:hyi-bng-01(config)#exit

RP/0/RSP0/CPU0:hyi-bng-01#sho running-config pool
