1.##################Add Profile และ register onu C6XX###################

1.1********************การ Add profile upload ********************

gpon
profile tcont UNLIMITED_UP type 4 maximum 1228800
profile tcont voip type 1 fixed 1536
profile tcont UNLIMITED_UP_Type3 type 3 assured 10240 maximum 819200
onu profile sip sipp42 registrar sipp42.totbb.net
profile tcont VPN-10M-IN type 1 fixed 10240
profile tcont VPN-30M-IN type 1 fixed 36860
!

1.2********************การ Add VLAN profile********************

gpon
onu profile sip sipp41 proxy sipp41.totbb.net
onu profile sip sipp41 registrar sipp41.totbb.net
onu profile sip sipp42 proxy sipp42.totbb.net
onu profile sip sipp42 registrar sipp42.totbb.net
onu profile vlan internet tag-mode tag cvlan 10
onu profile vlan voip tag-mode tag cvlan 20 pri 7
!

################### การ Register ONU ########################
1.************************register onu C6XX************************

# show pon onu uncfg
conf t
(config)# interface gpon_olt-1/2/1
(config)# (config-if-gpon_olt-1/2/1)# onu 1 type ZTE-F680 sn ZTEG090009E2 vport-mode manual
!

2.********************Interface gpon(ทำงานฝั่ง OLT)************************

2.1********************Internet************************

interface gpon_onu-1/1/1:1
name xxxxjxxxx
vport-mode manual
tcont 1 name internet profile UNLIMITED_UP
gemport 1 name internet tcont 1
vport 1 map-type vlan
vport-map 1 1 vlan 10

2.2********************Voice************************

tcont 2 name voip profile voip
gemport 2 name voip tcont 2
vport-map 1 2 vlan 20
$

3********************Interface vport(ทำงานฝั่ง OLT)************************

interface vport-1/1/1.1:1
service-port 1 user-vlan 10 vlan 3097

********************Voice************************

service-port 2 user-vlan 20 vlan 1500
$

4********************pon-onu-mng(ทำงานฝั่ง ONU)************************

4.1********************Bridge************************

pon-onu-mng gpon_onu-1/1/1:1
service internet gemport 1 vlan 10
vlan port eth_0/1 mode tag vlan 10
dhcp-ip ethuni eth_0/1 from-internet
$

4.2********************Route************************

pon-onu-mng gpon_onu-1/1/1:1
service internet gemport 1 vlan 10
wan-ip 1 ipv4 mode pppoe username xxxxjxxxx@fttxhome password xxxxxxxx vlan-profile internet host 1
wan-ip ipv4 ping-response enable traceroute-response enable
$

4.3********************Voice************************

pon-onu-mng gpon_onu-1/1/1:1
service voice gemport 2 vlan 20
voip protocol sip
voip-ip ipv4 mode dhcp vlan-profile voip host 2
voip-ip ipv4 ping-response enable traceroute-response enable
sip-service pots_0/1 profile sipp41 userid 0xxxxxxxx username 0xxxxxxxx password 0xxxxxxxx display-name 0xxxxxxxx
sip-service pots_0/2 profile sipp41 userid 0xxxxxxxx username 0xxxxxxxx password 0xxxxxxxx display-name 0xxxxxxxx
$

4.4********************VPN Route************************

(config)#gpon
(config-gpon)#onu profile ip pe-wan-vlan303 ipv4 gateway 10.6.3.253
(config-gpon)#onu profile vlan vlan303 tag-mode tag cvlan 303
$

interface gpon_onu-1/1/1:1
name 7742x0001 PEA
vport-mode manual
tcont 1 name vpn profile VPN-30M-IN
gemport 1 name vpn tcont 1
vport 1 map-type vlan
vport-map 1 1 vlan 303
$

interface vport-1/1/1.1:1
service-port 1 user-vlan 303 vlan 303
$

pon-onu-mng gpon_onu-1/1/1:1
service vpn gemport 1 vlan 303
wan-ip 1 ipv4 mode static ip-profile pe-wan-vlan303 ip-address 10.6.3.254 mask 255.255.255.252 vlan-profile vlan303 host 1

 

$

 

4.5********************Bridge Trunk************************

interface gpon_onu-1/1/1:1 name xxxxjxxxx
vport-mode manual
tcont 1 name internet profile UNLIMITED_UP
gemport 1 name internet tcont 1
vport 1 map-type vlan
vport-map 1 1 vlan 28
vport-map 1 1 vlan 350
vport-map 1 1 vlan 352

$

interface vport-1/1/1.1:1
service-port 1 user-vlan 28 vlan 28
service-port 2 user-vlan 10 vlan 350
service-port 3 user-vlan 30 vlan 352

$


pon-onu-mng gpon_onu-1/1/1:1
service mng gemport 1 vlan 28
service vpn gemport 1 vlan 10
service vpn1 gemport 1 vlan 30
vlan port eth_0/1 mode trunk
vlan port eth_0/1 vlan 28,10,30
dhcp-ip ethuni eth_0/1 from-internet

 


### หลังจาก static IP WAN ลงไปที่ ONU ให้ไปปิด NAT ที่เมนู Network => Routing(IPv4) => NAT โดยเอาเครื่องหมายถูกออก
